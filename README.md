# 🚗 Car Price Predictor using Machine Learning

## 📌 Project Overview

This project is a Machine Learning-based Car Price Predictor that estimates the price of a used car based on features like company, year, fuel type, and kilometers driven.

The dataset used in this project was cleaned and preprocessed to handle missing values, incorrect formats, and inconsistent data.

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit (for dashboard)

---

## 🔍 Features

* Data cleaning and preprocessing of real-world dataset
* Feature engineering using one-hot encoding
* Model training using Linear Regression
* Accuracy evaluation
* Interactive web dashboard using Streamlit

---

## 📊 Dataset Details

The dataset contains the following features:

* Company
* Year
* Price
* Kilometers Driven
* Fuel Type

---

## 🧠 Machine Learning Model

* Algorithm Used: Linear Regression
* Train-Test Split: 80/20
* Model Performance: Evaluated using R² score

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Car-Price-Predictor-ML.git
```

### 2. Install dependencies

```bash
pip install pandas numpy scikit-learn streamlit
```

### 3. Run the Notebook

Open `car_price.ipynb` and run all cells

### 4. Run the Dashboard

```bash
streamlit run app.py
```
